# Менеджер Паролей (Password Manager)

Современное кроссплатформенное desktop-приложение на **Kotlin Multiplatform** для безопасного хранения и управления паролями с шифрованием, генерацией надежных паролей и проверкой их безопасности.

## Описание проекта

Менеджер паролей — это полнофункциональное приложение для хранения учетных данных с акцентом на безопасность и удобство использования. Приложение написано на языке программирования **Kotlin** с использованием **Compose Multiplatform** для графического интерфейса. Оно предоставляет интуитивно понятный UI для управления паролями, их организации по папкам и тегам, а также автоматической генерации безопасных паролей.

## Основные возможности

- **Безопасное хранение паролей** — Все пароли шифруются алгоритмом AES-256 перед сохранением.
- **Генератор паролей** — Создание надежных паролей с настройкой длины и используемых символов.
- **Проверка надежности** — Визуальное выделение слабых паролей.
- **Организация** — Папки и теги для удобной категоризации записей.
- **Поиск** — Быстрый поиск по названию, логину или URL.
- **Импорт/Экспорт** — Возможность резервного копирования данных в JSON.
- **Контроль видимости** — Возможность скрывать и показывать пароли по требованию.
- **Современный UI** — Чистый интерфейс в стиле Material Design 3.

## Описание экранов

### 1. Экран входа (LoginScreen)
**Назначение:** Защита доступа к хранилищу.
**Функционал:**
- Ввод мастер-пароля.
- Генерация ключа шифрования на лету (SHA-256 хеширование мастер-пароля).
- Блокировка доступа к данным до успешной авторизации.

### 2. Главный экран (DashboardScreen)
**Назначение:** Основное рабочее пространство.
**Элементы:**
- **Сайдбар:** Навигация по папкам ("Все", "Работа", "Финансы") и фильтрация по тегам.
- **Список паролей:** Карточки с информацией (Название, Логин, скрытый Пароль, URL).
- **Верхняя панель:** Поиск и кнопки быстрых действий (Добавить, Импорт/Экспорт).
- **Индикаторы:** Предупреждения о наличии слабых паролей в базе.

### 3. Диалог добавления (AddPasswordDialog)
**Назначение:** Создание новых записей.
**Вкладки:**
- **Детали:** Ввод названия, логина, пароля и URL.
- **Генератор:** Встроенный инструмент для создания криптостойких паролей с настройками (цифры, спецсимволы, регистр).

### 4. Импорт/Экспорт (ImportExportDialog)
**Назначение:** Перенос данных.
- Экспорт базы в JSON формат (для резервного копирования).
- Импорт данных из внешних файлов.

## Технологический стек

- **Язык:** Kotlin 1.9.20+
- **UI Framework:** Compose Multiplatform (Desktop)
- **Material Design:** Material 3
- **Криптография:** Java Cryptography Architecture (AES/ECB/PKCS5Padding, SHA-256)
- **Сериализация:** Kotlinx Serialization (JSON)
- **Система сборки:** Gradle (Kotlin DSL)

## Безопасность
- **Шифрование:** Используется симметричное шифрование AES. Ключ шифрования генерируется динамически на основе мастер-пароля.
- **Хранение:** В памяти пароли хранятся в зашифрованном виде (passwordEncrypted), расшифровка происходит только в момент отображения.
